This data dashboard uses D3.js and DC.js to render the charts and Crossfilter to filter the data.

It is served with Python using the Flask web frame-work. It uses PostgreSQL to store the data and has been deployed with Heroku.

I followed Sahil Diwan's blog post Making a Flask app using a PostgreSQL database and deploying to Heroku for my app structure, for help setting up my database and to aid in deploying to Heroku.
